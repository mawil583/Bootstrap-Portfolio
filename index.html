About Me

Hello! My name is Michael and I am a full-stack software engineer. Skills of mine include HTML5, CSS3, JavaScript (ES6), Command Terminal, Bootstrap, jQuery, Node.js/Express, SQL/Sequelize, Mongo/Mongoose, React, and NPM packages.

This website was created from scratch using Bootstrap, which is a CSS framwork. It works by referencing a CSS file in the head tag of HTML. This CSS file is prewritten, so it's kind of like copy-pasting code that another programmer has developed. If that seems too easy, you're right. It is. But, in addition to copy-pasting, you have to make sure you have a good understanding of their grid system in order to be proficient at Bootstrap. One of the great things about Bootstrap is that it comes with a pre-built responsiveness that makes the content respond to a resizing of the browser window. This also makes it compatible with mobile phones.
